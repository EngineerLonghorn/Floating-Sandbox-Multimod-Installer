Welcome!

This is the Floating Sandbox Multimod Installer, made to make installing several mods for Floating Sandbox simpler, quicker, and easier, using Inno Setup to compile all known mods in the Floating Sandbox Designer's Studio Discord Server into one customizable installer.

All you need to make or edit the installer is Inno Setup by Jordan Russell, and Floating Sandbox by Gabriele Giuseppini.
